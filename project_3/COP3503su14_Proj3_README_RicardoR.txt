project3 README file

To compile:
compile: g++ -c tinyxml2.cpp -o tinyxml2
input make on the terminal.
Be sure to have the tinyxml2.o in the same folder as the make file

To run:
input ./project_3

Known to compile with (from g++ -v): 
gcc version 4.6.4 (Ubuntu/Linaro 4.6.4-1ubuntu1~12.04) 

No special resources needed. 
No configuration needed.
Hit ENTER after every command.
